README:
Two Classes have main methods:
kdTreeViewer: Contains graphics interface for adding points to a KD Tree
PerformanceTrial: Gives the performance results for the add function and the nearestNeighbor query

Other classes:
DrawArea: used to implement graphics interface
Model: used to implement graphics interface
KDTree: The KD Tree itself
KDNode: KDNode in a KDTree
Region: The region a KDNode partitions
Orientation: is an enum
